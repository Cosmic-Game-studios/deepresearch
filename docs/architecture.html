Every experiment produces data that flows into multiple stores. The experiment log is the source of truth. Everything else can be rebuilt from it.

The experiment log (experiments.jsonl) is append-only and the single source of truth. If any other file gets corrupted, it can be rebuilt from the log. The knowledge base, dependencies, and strategy state are derived views that accelerate decision-making.
